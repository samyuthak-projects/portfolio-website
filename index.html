Building My First AI Chatbot

20th Mar 2026 — I built a Python chatbot that can respond to user input using a machine learning model. I used a Naive Bayes classifier with CountVectorizer to classify messages into intents.

One challenge I faced was understanding how text is converted into numbers for the model. I learned that machine learning models don’t understand words directly, so tools like vectorizers are used to represent text numerically.

This project introduced me to natural language processing and gave me insight into how real chatbots work.
